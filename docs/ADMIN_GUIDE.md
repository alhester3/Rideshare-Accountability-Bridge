# Admin Guide

## Admin Experience
Admins manage user accounts, review and resolve appeals, monitor audit logs, and track fairness metrics.

## Core Admin Tools
- User management panel
- Appeal queue and resolution workflows
- Audit and compliance center
- Fairness monitoring and metric summaries
- Bias/flag review paths
